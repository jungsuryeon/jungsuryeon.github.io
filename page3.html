게임 소개

총 4개의 스테이지로 구성
등장인물 : 시민(들), 좀비(들), 마동석
기차길이 -> 마동석 체력 -> 확률 순 입력(잘못 입력되면 재입력 가능, 4개의 스테이지 중 처음 1번만 입력)

스테이지 1
시민 : (100-p)% 확률로왼쪽으로 1칸이동하거나(어그로 증가), p% 확률로제자리에대기(어그로 감소) 이동가능한턴이면시민과마동석중어그로수치가높은쪽으로이동. 어그로수치가 같으면시민쪽으로이동
좀비 : 1턴은 p% 확률로왼쪽으로 1칸 이동하거나, (100-p)% 확률로 제자리에대기, 2턴마다 이동가능
마동석 : stay-제자리대기 (어그로 감소)/ left-왼쪽으로 이동 (어그로 증가)
마동석 행동 : rest - 휴식(어그로 감소, 체력증가)/ provoke-도발(어그로 최대로 증가)/pull-붙들기(어그로 2 증가, 체력 1감소)
시민이 맨 왼쪽 => 우승
좀비가 옆에 있으면 => 실패
마동석 체력이 0이면 => 실패

스테이지 2
스테이지 1과 동일하지만 빌런 추가
빌런과 시민이 인접하면 30% 확률로 위치 교환

스테이지 3
스테이지 1과 달리 시민이 랜덤 숫자로 여러 명 생성
좀비에게물린시민은삭제

스테이지 4
스테이지 3과 동일하지만 좀비에게 물린 시민은 좀비로 변함
